# OFT Token Launchpad

## В одном предложении

No-code лаунчпад для создания и бриджинга кросс-чейн токенов на базе LayerZero V2 OFT стандарта.

## Проблема

Создание кросс-чейн токенов требует глубоких знаний Solidity, ручной настройки контрактов и сложной конфигурации peer-соединений между сетями. Простого инструмента для создателей и комьюнити не существует.

## Решение

OFT Token Launchpad позволяет любому задеплоить OFT-совместимый токен на любой поддерживаемой сети в один клик, а затем бриджить его на другие сети через LayerZero — всё из современного веб-интерфейса.

## Ключевые возможности

- **Создание токена в один клик** — деплой полноценного OFT-токена через фабрику. Знание Solidity не требуется.
- **Кросс-чейн бриджинг** — перевод токенов между сетями через LayerZero V2 с оценкой комиссии в реальном времени.
- **Дашборд токенов** — просмотр всех созданных токенов, балансов и ссылок на эксплорер для каждой сети.
- **Мультичейн** — Sepolia, Mumbai, Arbitrum Sepolia, Optimism Sepolia, Base Sepolia.
- **Factory-паттерн** — все токены отслеживаются on-chain: по создателю, с пагинацией и индексацией событий.

## Технологический стек

| Слой | Технология |
|------|-----------|
| Смарт-контракты | Solidity 0.8.22, Hardhat, OpenZeppelin |
| Кросс-чейн | LayerZero V2 Endpoint, OFT Standard |
| Фронтенд | React 18, TypeScript, Vite, TailwindCSS |
| Кошелёк | RainbowKit, Wagmi 2, Viem |
| Тестирование | Hardhat + Chai (26 тестов) |

## Архитектура

```
Пользователь → Фронтенд (React) → Wagmi/Viem → Смарт-контракты
                                                      │
                                                OFTFactory.sol
                                                      │
                                                LaunchpadOFT.sol
                                                      │
                                             LayerZero Endpoint V2
                                                      │
                                              Кросс-чейн сообщение
                                                      │
                                             OFT на целевой сети
```

### Смарт-контракты

**OFTFactory.sol** — фабрика, деплоящая OFT-токены и отслеживающая все деплои on-chain:
- `createToken(name, symbol, initialSupply)` — создать новый OFT
- `createTokenWithDelegate(name, symbol, initialSupply, delegate)` — создать с кастомным LZ-делегатом
- `getTokensByCreator(address)` — список токенов по создателю
- `getDeployedTokens(offset, limit)` — пагинированный список
- `isFactoryToken(address)` — проверка происхождения токена

**LaunchpadOFT.sol** — OFT-токен с метаданными:
- Наследует LayerZero OFT для нативных кросс-чейн трансферов
- Хранит `creator`, `createdAt`, `initialSupply` как immutable
- 18 decimals, стандартный ERC-20 интерфейс

### Фронтенд

Три основных раздела:
1. **Create Token** — форма с именем, символом, supply; деплой через фабрику
2. **Bridge** — выбор токена, сети назначения, суммы; расчёт комиссии и отправка через LZ
3. **My Tokens** — список токенов пользователя с балансами и ссылками на эксплорер

## Как используется LayerZero

1. **OFT Standard** — каждый токен, созданный через лаунчпад, является нативным LayerZero OFT, а не обёрткой
2. **Endpoint V2** — фабрика передаёт адрес LZ Endpoint каждому токену при деплое
3. **Peer Connections** — токены на разных сетях связываются через `setPeer()` для двунаправленного обмена
4. **Cross-Chain Send** — бриджинг вызывает `send()` OFT, который использует LayerZero messaging
5. **Fee Quoting** — UI вызывает `quoteSend()` для отображения точной комиссии перед бриджингом

## Покрытие тестами

26 тестов, покрывающих:
- Деплой и валидацию фабрики
- Создание токенов и эмиссию событий
- Трекинг по создателям и разделение токенов
- Пагинацию (крайние случаи: offset за пределами, неполные страницы)
- Функционал делегата
- Метаданные OFT и операции ERC-20
- Крайний случай с нулевым supply

## Категории для награды

### Best Use of OFT ($2,000)
Нативная имплементация OFT с полноценным лаунчпад UX — токены являются OFT с момента создания, а не адаптерами или обёртками.

### Best Use of LayerZero ($3,000)
Полная интеграция LayerZero V2: Endpoint V2, OFT-стандарт, настройка peer-соединений, кросс-чейн messaging, расчёт комиссий — всё доступно через отполированный UI.

## Демо-сценарий

1. Подключить кошелёк (MetaMask через RainbowKit)
2. Создать токен на Sepolia — "MyToken", "MTK", 1 000 000 supply
3. Посмотреть токен во вкладке "My Tokens" с балансом и ссылкой на эксплорер
4. Настроить peer-соединения между Sepolia и Arbitrum Sepolia
5. Забриджить 1 000 MTK из Sepolia в Arbitrum Sepolia
6. Отследить трансфер на LayerZero Scan
